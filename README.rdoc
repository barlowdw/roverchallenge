= roverchallenge

A ruby gem that implements the Mars Rover Challenge as specified at https://code.google.com/archive/p/marsrovertechchallenge/

== Setup

* gem install bundler
* bundle install

== Build

* rake build
* rale install

== Run

* ./bin/rover_challenge < spec/files/test.txt
* Any file can be used as the input instead of spec/files.test.txt.
